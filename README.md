# Objetivo do projeto: 
a Opuline, uma empresa fictícia que vende cosméticos e atua no ramo da beleza. A Opuline ainda não é orientada a dados, mas quer fazer essa transformação para ser parte da cultura Data Driven.<br>
O desafio é entender quais são os gráficos e visuais mais indicados para cada tipo de análise desejada. 
# Contexto do negocio:
Com as analises realizadas os gestores da Opuline podem definir estrategias para aumentar as vendas nacionais, definir novas metas com base na previsão de faturamento e identificar produtos e categorias com maior volume de vendas para promoções ou ajustes de preço.
# Fonte do dados:
Dados ficticios / simulados <br>
Formato dos carquivos Excel / SQL <br>
# Modelagem de dados:
Modelo estrela <br>
Tabelas dimensão: dCategoriasProdutos, dMarcas, dMetasMensal e dProdutosFinais <br>
Tabela fato : fPedidos
# Principais visuais:
Cartões: Faturamento, quantidade vendida
Barras: Faturamento de vendas por região e produto
linhas: Receita ao longo do tempo com previsão de crescimento
Mapa: Quantidade de vendas por região
